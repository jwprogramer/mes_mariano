# Mês Mariano

Este repositório contém um projeto dedicado ao **Mês Mariano** da Pároquia de Santo Antônio, em Serrinha/RN, um período tradicional da Igreja Católica dedicado à devoção a Maria. 

## 📌 Objetivo

Esse projeto tem como objetivo de apresentar todos os noiteiros do Mês Mariano no ano de 2026, facilitando a consulta e participação das famílias convidadas. 

## 📂 Estrutura do Repositório

A organização pode variar conforme o desenvolvimento do projeto, mas geralmente inclui:

- `assets/` — imagens
- `README.md` — esta documentação
